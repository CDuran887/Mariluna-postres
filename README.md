# Mariluna Postres - Sistema de Gestión de Inventario

Este es un proyecto de **Programación Orientada a Objetos (POO)** desarrollado en Java para la gestión de un catálogo de productos en un entorno de repostería. El sistema permite administrar productos y gestionar las interacciones básicas entre un administrador y el inventario.

## Características

- **Gestión de Productos:** Creación y actualización de información de postres (nombre, precio, stock, categoría).
- **Control de Acceso:** Clase Administrador para la supervisión del catálogo.
- **Diseño Modular:** Separación clara entre entidades y lógica de aplicación.
- **Arquitectura Limpia:** Aplicación de principios SOLID (especialmente Responsabilidad Única).

## Tecnologías Utilizadas

* **Lenguaje:** Java 21 (OpenJDK)
* **Gestor de Dependencias:** Maven
* **IDE:** Visual Studio Code
* **Pruebas:** JUnit 4.13.2
